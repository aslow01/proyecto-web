Hacé una web tipo dashboard administrativo

No una sola página llena de botones.

La dividiría así:

1. Inicio

Una pantalla principal con tarjetas grandes:

Movilidades
Objetivos
Partes diarios
Novedades
Reportes
Configuración

Eso hace que sea más fácil de usar para cualquiera de la empresa.

2. Módulo “Movilidades”

Acá pondría todo lo relacionado a vehículos, choferes o unidades.

Subsecciones:

Por provincia
Mendoza
San Juan
Santa Cruz
Listado general
patente
descripción
estado
ubicación
objetivo asignado
última novedad
Filtros
provincia
objetivo
estado
fecha

Esto te va a servir mucho más que tener todo mezclado.

3. Módulo “Objetivos”

Acá entran clientes, proyectos o zonas de trabajo.

Ejemplo:

Barrick
Minera
Urbano
Gema / Gemera
Otros objetivos

Y dentro de cada objetivo mostrar:

unidades asignadas
choferes
partes diarios
novedades
estado operativo

Así cada objetivo funciona casi como una ficha propia.

4. Módulo “Partes diarios”

Este módulo puede ser de los más importantes.

Cada parte diario podría tener:

fecha
provincia
objetivo
unidad
chofer
km inicial/final
combustible
observaciones
desperfectos
estado del día

Y estaría bueno que se pueda:

cargar
editar
buscar por fecha
exportar
5. Módulo “Novedades”

Para anotar cosas importantes:

roturas
mantenimientos
cambios de unidad
retrasos
incidentes
observaciones operativas

Y que tenga prioridad visual:

roja = urgente
amarilla = pendiente
verde = resuelto

Eso hace que la página sea útil de verdad en logística.

Cómo la organizaría visualmente
Opción más recomendable: menú lateral

Un diseño con:

Barra lateral izquierda
Inicio
Movilidades
Objetivos
Partes diarios
Novedades
Reportes
Usuarios
Parte superior
buscador
usuario logueado
fecha
alertas
Contenido central

Tablas, tarjetas, filtros y formularios.

Eso se ve más profesional y es más cómodo que un menú arriba si va a tener muchos apartados.

Qué estilo visual te recomiendo

Para una empresa logística, haría algo:

moderno
limpio
serio
rápido de entender

Colores recomendados:

azul oscuro o gris grafito para menú
blanco o gris claro para fondo
azul/celeste para botones principales
rojo/amarillo/verde para estados

Evitaría una web muy cargada o con demasiados colores.

Cosas que te conviene agregar

Tu idea está buena, pero yo le sumaría estas funciones:

Buscador general

Para encontrar rápido:

una movilidad
un objetivo
una patente
una novedad
Filtros

Muy importante si va a haber muchos datos.

Estados visuales

Por ejemplo:

Activo
En taller
Fuera de servicio
Pendiente
Historial

Que cada movilidad tenga historial de:

novedades
partes diarios
mantenimiento
gastos
Login de usuarios

Si la usan varias personas:

administrador
supervisor
operador

Así no todos pueden modificar todo.

Qué tecnología usaría

Si querés algo moderno y escalable:

Frontend: HTML, CSS, JavaScript
mejor todavía: React
